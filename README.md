# Login-Form
This project is a responsive Login and Registration Form built using HTML, CSS, and JavaScript. It provides a smooth user experience with animated transitions for switching between forms and works well across different screen sizes.

## 🚀 Features

- ✅ Login & Registration system
- ✅ Smooth sliding animations
- ✅ Responsive design (mobile-friendly)
- ✅ Social media icons integration
- ✅ Clean and modern UI
- ✅ Easy to use interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome (for icons)

---

## 📂 Project Structure

login-form/
│── login.html
│── login.css
│── login.js
│── README.md


---

## 🎯 How It Works

- Click **Register** → Registration form appears  
- Click **Login** → Login form appears  
- JavaScript toggles the UI using class changes
